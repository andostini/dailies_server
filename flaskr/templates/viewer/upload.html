File Upload

Upload Proxies and Stills with this form. Proxies will be assigned to a clip in the library by their filename (always exlcuding extension). If a proxy doesn't find a match in a the library, it will be added as a clip to the library with empty metadata. You can then use our meta editor or upload a metafile later. Stills will be assigned to clips by their filename. You can add multiple stills to one clip, if you name you still "clipname_X", replacing X with a number (hint: use timecodestamp for X in Silverstack to have stills automatically listed chronologically).
Duplicate filenames will be overwritten
